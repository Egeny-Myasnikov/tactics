# Project: _"TacTics"_ | Проект _"TacTics"_

## Backend part | Серверная часть

### Description:

> TacTics app - this is a specialized online platform that connects hockey trainers with students and their representatives. The project solves the problem of finding qualified hockey trainers, taking into account their specialization, pricing policy, and geographical location.

### Описание:

> Приложение TacTics - это специализированная онлайн-платформа, соединяющая хоккейных тренеров с учениками и их представителями. Проект решает проблему поиска квалифицированных тренеров по хоккею с учетом специализации, ценовой политики и географического расположения.
